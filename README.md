# A Ralph language parser in TypeScript written

## Goal

- Generate AST

## Build

- install dep

```shell
make dep
```

- build

```shell
make
```

or

```shell
npm run antlr4ts
```

or

```shell
yarn run antlr4ts
```

## Install use
[npm package](https://www.npmjs.com/package/ralph-parser-ts)
```shell
npm i ralph-parser-ts
```
or
```shell
yarn add ralph-parser-ts
```
